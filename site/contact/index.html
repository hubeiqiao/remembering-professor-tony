What this contact path is for

The memorial domain keeps the tribute focused, but it does not try to replace Joe Hu's main identity surface. If you want to follow up about the site, ask about authorship, or understand the broader context of Joe's work, the main site is the right place to continue.

This page exists so that search engines and answer engines can see an explicit contact route without turning the tribute itself into a generic profile page. That separation keeps the memorial readable while still giving readers a clear next step.

For a broader view of Joe Hu's public work, visit hubeiqiao.com. For the long-form reflection connected to this tribute, use the essay linked above.

Good contact use cases include questions about the authorship of the tribute, requests to understand the public scope of the project, or follow-up messages related to the memorial itself. The main site is also where Joe Hu's broader work and current identity are maintained.

This page is intentionally simple, but it serves an important role for machine-readable publishing. It makes the authorship chain explicit and gives crawlers a stable contact endpoint without forcing them to infer identity only from the narrative article.

In other words, this page is part of the trust layer of the site. It tells both people and machines where the author identity actually lives, which reduces ambiguity and makes the memorial easier to understand as a real work with a named public author.

This memorial keeps identity local to the project while routing direct contact to hubeiqiao.com.
