# Research project planning for AI-enabled English speaking skills improvement

Tue, 02 Sept 25

### Research Project Framework - 5104 Course

- Modified research question: “How can AI enable improvement in English speaking skills?”
  - Removed personalization constraint - may be one answer but not the only one
  - Broader scope allows for multiple solution approaches
- Deliverable breakdown for 13-week course:
  - Literature review on AI-enabled English speaking improvement (70%)
  - Prototype development and validation (20%)
  - Process adoption methodology (10%)
- Literature review structure:
  - Two-part approach: academic articles + real-world use cases
  - Include company examples and implementation methods
  - Focus on practical applications, not just academic theory
- Prototype phase:
  - Build functional prototype ready for testing
  - Stop at development stage - no user testing required for course
  - May extend to user testing for capstone project

### Course Integration Strategy

- Complement with 5110 (technical course) for comprehensive coverage
- 5104 focuses on research and feasibility analysis
- Capstone project will handle deployment phase
- Client engagement optional for 5104 but valuable for capstone
  - Mentioned potential client: language teacher with small student base
  - Goal: scale teaching method to thousands/millions of users

### Career Development Objectives

- Three key competencies employers seek:
  - AI value creation - demonstrate ability to generate business value with AI
  - Project formulation and implementation - end-to-end project management
  - Convincing others - value proposition development and stakeholder buy-in
- Students achieving high salaries demonstrate these three capabilities
- Focus on practical impact: revenue generation, cost reduction, or enabling new capabilities
- Technology knowledge less important than ability to create measurable business value
